This project focuses on Comment Toxicity Detection using Natural Language Processing (NLP) techniques.
The goal is to automatically classify online comments as toxic or non-toxic, helping create safer and more positive online communities.

This is a learning-based guided project completed as part of my NLP learning journey using a YouTube tutorial.

🚀 Features

Text preprocessing and cleaning

Tokenization and text vectorization

Toxic comment classification

Deep learning based NLP model

Prediction on custom user comments

🛠️ Technologies Used

Python 🐍

TensorFlow / Keras

NLP Techniques

NumPy & Pandas

Scikit-learn

Jupyter Notebook
